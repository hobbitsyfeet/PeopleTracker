\doxysubsection{Detailed Description}
The {\ttfamily Multi\+Tracker} class is a Python class that represents a multi-\/object tracker and contains various attributes and methods for tracking objects. 


\begin{DoxyParams}{Parameters}
{\em tab} & is a designated tab in the UI which will populate provided values. \\
\hline
{\em colour} & is a 255/255/255 RGB value that defines a tracking colour for easy distinction.\\
\hline
\end{DoxyParams}
Many values are retrieved from {\ttfamily tab} defined in Tracker\+Tab.\+py which is generated in qt\+\_\+dialog.\+py\textquotesingle{}s App.

\begin{DoxyVerb}Parts the times into segments based on the distance between tracked frames.
    time_data is the entire time of the object tracked.
    segment_size is the distance between frames that should be considered as one segment. Default is 10 seconds at 30fps.
    min_segment is the minimum size of segment allowed. If it is too small, it will be ignored. Defualt is 1 second (30 frames).
