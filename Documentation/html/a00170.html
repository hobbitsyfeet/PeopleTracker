Detailed Description

The MultiTracker class is a Python class that represents a multi-object tracker and contains various attributes and methods for tracking objects.

Parameters
tabis a designated tab in the UI which will populate provided values.
colouris a 255/255/255 RGB value that defines a tracking colour for easy distinction.

Many values are retrieved from tab defined in TrackerTab.py which is generated in qt_dialog.py's App.
